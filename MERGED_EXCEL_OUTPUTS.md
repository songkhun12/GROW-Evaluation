# Generated merged files

The merge can be generated locally in each grade folder by running:

```bash
python merge_at_risk_reports.py
```

Because binary Excel files are not supported in the PR diff/review flow, the committed reviewable outputs are CSV files. The script also creates local `.xlsx` workbooks with the same base names, but those Excel files are intentionally ignored by git.
