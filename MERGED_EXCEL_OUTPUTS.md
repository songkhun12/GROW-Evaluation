# Generated merged Excel files

The merged Excel files are generated locally in each grade folder by running:

```bash
python merge_at_risk_reports.py
```

They are intentionally ignored by git because binary Excel files are not supported in the PR diff/review flow. After running the script, the files are located at:

## Kindergarten
- `Kindergarten/JES Atlas Report - Kindergarten - Merged At Risk.xlsx`
- `Kindergarten/JMTES Atlas Report - Kindergarten - Merged At Risk.xlsx`
